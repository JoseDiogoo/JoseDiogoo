## 🚀 Projeto de Destaque

### 🔎 Vec4ensics  
**Procura Semântica e Bases de Dados Vetoriais para Investigação Digital Forense**

Aplicação desenvolvida com o objetivo de ultrapassar as limitações da pesquisa lexical tradicional, permitindo **pesquisa semântica baseada em similaridade vetorial**.

### Principais Componentes Técnicos

- Geração de embeddings de texto
- Indexação vetorial com FAISS
- Pesquisa por similaridade semântica
- Pipeline de ingestão de documentos (PDF, DOCX, XLSX)
- Arquitetura modular orientada a componentes
- Implementação em Python

### Tecnologias Utilizadas

- Python
- Processamento de Linguagem Natural (NLP)
- FAISS
- Manipulação e processamento de ficheiros
- Estruturas de dados vetoriais
